# 红尘识相 · Windows 桌面端

红尘识相是一款面向相学从业者的辅助工具，内含相历、采相、时区流动图、盘时辰四大功能，帮助相师提升排盘与研判效率。

## 功能列表

- **相历**：查阅相学历法
- **采相**：一键采集与排盘
- **时区流动图**：可视化时区流动
- **盘时辰**：时辰排盘工具

## 下载与安装

1. 下载地址：[https://github.com/xiaoyaohe777/hongchenshixiang]
2. 下载压缩包后，将**所有文件**解压到桌面端同一目录（不要跳过任何文件，压缩包内的配置文件是程序运行必需项）
3. 打开 `红尘识相.exe`即可使用

⚠️ **注意**：解压出的所有文件请保留在同一目录下，不要删除或移动其中任何文件（包括配置文件），否则程序将无法正常运行。

## 系统要求

Windows 10 及以上

## 常见问题

- **打开后闪退/无法启动？**：请确认压缩包内所有文件是否完整解压到同一目录，未缺失、未单独移动配置文件
- **和网页版功能一样吗？**：一样的
