红尘识相 · Windows 桌面端 本次更新（v1.2.0）
红尘识相是一款面向相学从业者的辅助工具，内含万年历、采相、时辰流动图、实验田四大功能，帮助相师提升排盘与研判效率。

下载与安装

1. 下载地址：[https://github.com/xiaoyaohe777/hongchenshixiang/releases]
2. 下载后，点安装，一路next就可以了，C盘不够用的，系统性洁癖的，请安装到D盘
3. 打开 `红尘识相` 即可使用

功能列表

* 万年历：公历·农历·相历三历互查，年月属相按立春、节气精确分界演算
* 采相：一键采集与排盘，输入相主信息即可极速计算采相分数
* 时辰流动图：可视化十二时辰流动规律，辅助精准判断相主见天光时辰
* 实验田：更多功能实验中，持续更新

本次更新（v1.2.0）

* 精准分界：年属相、月属相改为按"立春""节气"精确演算，交节当天即变，不再延后到第二天，排盘更贴近传统命理规则
* 节气提醒：打开万年历，逢节气当天自动标注（如"今日大暑"），并提示距下一节气还有几天
* 玄门神谱：逢尊神诞辰自动提醒，附点灯指引，为相主祈福添一份仪式感
* 月历手感升级：万年历主页支持左右滑动切换月份，长按日期可快速预览当日信息
* 时辰流动图优化：操作提示更清晰，时辰标注统一规范（如"狗时""猪时"）

⚠️ 注意：安装的所有文件请保留在同一目录下，不要删除或移动其中任何文件（包括配置文件），否则程序将无法正常运行。
